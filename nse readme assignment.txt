
 Student Record Manager

 Python Mini Project



  Project Title

Student Record Manager



  Project Description

The **Student Record Manager** is a simple, menu-driven Python console application designed to manage student details efficiently.
It allows users to **add, view, search, update, and delete student records** using basic Python concepts.
This project is suitable for beginners and follows the guidelines of a Python mini project.



  Objective

* To understand how real-world data can be managed using Python
* To practice core Python concepts like lists, dictionaries, functions, loops, and file handling
* To build a simple and interactive console-based application



  Features

* Add new student records
* Display all student records
* Search student details using Student ID
* Update existing student information
* Delete student records
* Menu-driven and user-friendly interface
* Data stored using Python data structures



  Concepts Used

* Python Basics
* Lists and Dictionaries
* Functions
* Conditional Statements
* Loops
* Exception Handling (`try` & `except`)
* File Handling (optional – if used)



  Instructions to Run the Program

1. Make sure **Python 3.x** is installed on your system
2. Download or copy the project file (e.g., `student_record_manager.py`)
3. Open Command Prompt / Terminal
4. Navigate to the project folder
5. Run the program using:

```bash
python student_record_manager.py
```



  Input Guidelines

* Student ID should be a **number**
* Name should contain **alphabetic characters only**
* Age should be a **valid number**
* Department name should be entered correctly
* Follow on-screen instructions carefully



  File Handling Details (If Applicable)

* Student data is stored using Python lists and dictionaries
* (Optional) Data can be saved to a `.txt` or `.csv` file for persistence
* File is read and updated whenever changes are made



  Sample Data Fields

* Student ID
* Student Name
* Age
* Department
